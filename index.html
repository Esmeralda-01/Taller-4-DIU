Definición




Bootstrap es un framework que ofrece un conjunto de herramientas de
código abierto para el front-end de un sitio web, se combina CSS
y JavaScript para forma a los elementos de una página HTML y
varios otros componentes que están preconstruidos.
Este permite al desarrollador crear sitios web rápidamente
implementando el diseño que desee, evitando así la tediosa tarea
de escribir amplios códigos en CSS.

Características

Fácil de manejar

Ofrece un soporte en HTML5 y CC3. Así, los usuarios pueden hacer uso de él con una gran flexibilidad

Tiene mejoras en el sistema de grid

Permite insertar imágenes responsive de una forma muy fácil

Se integra muy bien con las principales librerías Javascrip

Se matiene al tanto de tendencias de diseño

Ejemplos de uso

  • Fragmentos

    Patrones comunes para crear sitios y aplicaciones que se basan en componentes y utilidades existentes con CSS personalizado y más.

  • Componentes personalizados

    Componentes y plantillas completamente nuevos para ayudar a las personas a comenzar rápidamente con Bootstrap y demostrar las mejores prácticas para agregar al marco.

  • Estructura

    Ejemplos que se centran en la implementación de usos de componentes integrados proporcionados por Bootstrap.

  • Barras de navegación

    Tomando el componente de barra de navegación predeterminado y mostrando cómo se puede mover, colocar y extender.
